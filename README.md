# 🚀 Primetrade Backend Assignment

## 📌 Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- React.js (Frontend)

---

## 🔐 Features

- User Registration & Login
- JWT Authentication
- Role-Based Access (User/Admin)
- CRUD APIs (Tasks)
- Secure Password Hashing
- Error Handling & Validation

---

## 📦 API Endpoints

### Auth

- POST /api/auth/register
- POST /api/auth/login

### Tasks

- POST /api/tasks
- GET /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id

---

## ▶️ Setup Instructions

### Backend

```bash
cd backend
npm install
npm run dev

```
